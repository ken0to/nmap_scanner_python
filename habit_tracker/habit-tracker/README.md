# Habit Bloom

An Expo Router + React Native TypeScript habit tracker with local AsyncStorage persistence.

## Features

- Create and edit habits
- Daily or weekly frequency with a target count
- Two-column home grid with completion controls
- Seven-day progress squares on each habit card
- Detail screen with monthly calendar, streak, duration, and target stats
- Mark past days from the calendar
- Archive habits
- Dark iOS-style UI with muted gradient cards and a bottom sheet action menu

## Run

```bash
npm install
npm run start
```

Then open the app in Expo Go, an iOS simulator, Android emulator, or web.
